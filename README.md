#html_preview

A React app that gives you an instant preview of any HTML code.

## Building/running
Run
```shell
$ npm install
```
and then
```shell
$ gulp
```
to open it up in a web server, or
```shell
$ gulp deploy
```
to build the files (found in dist/)
